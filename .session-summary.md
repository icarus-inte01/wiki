- Deploy step: `actions/deploy-pages@v4`

## Quick Start
```bash
cd ~/doc/wiki
source .venv/bin/activate    # or: python3 -m mkdocs serve
mkdocs serve                 # http://127.0.0.1:8000
```

To publish:
```bash
git add .
git commit -m "message"
git push                     # Actions auto-deploys
```
